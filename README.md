## 📖 Sobre o Projeto

Portfólio web responsivo desenvolvido para apresentar minha trajetória como Cientista de Dados e Desenvolvedor Full-Stack. O site foi construído do zero utilizando HTML5, CSS3 e JavaScript puro, com foco em experiência do usuário, performance e design moderno.

### 🎯 Objetivo

Criar uma plataforma digital profissional que demonstre minhas competências técnicas através de:
- Apresentação clara de projetos e certificações
- Interface intuitiva e visualmente atraente
- Código limpo e bem estruturado
- Responsividade total em todos os dispositivos

## ✨ Funcionalidades

### 🎨 Interface & UX
- ✅ **Tema Claro/Escuro** - Alternância dinâmica com persistência (localStorage)
- ✅ **Scroll Animations** - Elementos aparecem suavemente ao rolar a página
- ✅ **Menu Responsivo** - Navegação adaptável para mobile e desktop
- ✅ **Smooth Scroll** - Transições suaves entre seções

### 🎪 Componentes Interativos
- ✅ **Carrossel de Projetos** - Navegação intuitiva com botões e indicadores
- ✅ **Carrossel de Certificados** - Exibição organizada de qualificações
- ✅ **Ticker de Habilidades** - Animação horizontal infinita de tecnologias
- ✅ **Botões CTA** - Call-to-actions para GitHub e sites externos

## 🚀 Destaques Técnicos

### Performance
- ⚡ **Carregamento otimizado** - Imagens comprimidas e lazy loading
- ⚡ **CSS minificado** - Redução de tamanho para carregamento rápido
- ⚡ **JavaScript eficiente** - Event delegation e debouncing

### Acessibilidade
- ♿ **Semântica HTML** - Tags apropriadas (header, nav, section, article)
- ♿ **Contraste adequado** - WCAG 2.1 Level AA
- ♿ **Navegação por teclado** - Suporte completo

### SEO
- 🔍 **Meta tags** otimizadas
